# CertifyX

Modern digital credential management platform built with React.

## Features

- Secure Login Interface
- Dashboard Overview
- Credential Hub (Card & List View)
- Credential Details Page
- User Profile Management
- Fully Responsive Design

## Tech Stack

- React
- React Router DOM
- useState Hook
- Bootstrap 5
- SCSS

## Live Demo

https://your-vercel-link.vercel.app

## Project Purpose

CertifyX is a modern credential management platform concept designed to help users organize, access, and manage professional and academic credentials through a clean and responsive user interface.
